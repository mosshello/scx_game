# 版权与免责声明 技术规格书

## 目标
- 新增一份精简、清晰且不以格式条款排除法定责任的版权与免责声明，并从站点页脚提供稳定入口。

## 影响范围
| 文件 | 原因 |
|---|---|
| `disclaimer.html` | 承载完整的版权、收费说明、使用规范、侵权通知和责任边界 |
| `index.html` | 将现有免责声明提示改为独立页面入口 |
| `game.html` | 在游戏详情页页脚补充声明入口 |
| `css/style.css` | 提供声明页桌面端与移动端样式 |
| `.ai/project.md` | 记录项目状态和验证结果 |

## 实施步骤
1. 新增语义化的《版权与免责声明》页面，避免使用“网络公开即可使用”“整理费可替代授权”“本站概不负责”等表述。
2. 明确第三方权利归属、收费内容边界、用户合法使用义务、侵权通知材料和及时处理机制。
3. 明确声明不排除网站依法应承担的责任，也不影响消费者法定权利。
4. 将首页和详情页页脚链接到声明页面。
5. 增加响应式页面样式，并验证链接、HTML 结构和移动端布局。

## 验证方式
- 检查 `index.html` 与 `game.html` 的声明链接均指向 `disclaimer.html`。
- 搜索页面中是否不存在绝对免责或将公开来源等同于授权的高风险表述。
- 通过本地 HTTP 服务检查三个页面均返回 200。
- 检查 390px 移动端和桌面端声明页无横向溢出。

## 回滚方案
- 删除 `disclaimer.html` 及其专用样式，恢复两个页脚原链接内容。
