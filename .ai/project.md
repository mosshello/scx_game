# 66GM 项目上下文

## 项目定位
66GM 是一个展示、检索并引导体验或解锁免广告小游戏的静态网站。

## 技术约束
- 技术栈：原生 HTML、CSS、JavaScript 与 JSON 数据。
- 架构边界：页面结构位于 HTML，视觉样式位于 `css/`，交互与数据渲染位于 `js/`，游戏数据位于 `data/`。
- 仓库约束：`js/scraper.js` 是本地私有同步工具，必须保持 Git 忽略，不得提交到公开仓库。
- 默认语言：规格书、恢复方案、变更日志、审查结论、交接说明和新增注释默认中文。

## 关键路径
| 模块 | 路径 | 职责 |
|---|---|---|
| 首页 | `index.html` | 首页结构与内容入口 |
| 首页样式 | `css/style.css` | 首页布局、视觉与响应式样式 |
| 首页脚本 | `js/app.js` | 游戏数据、滚动游戏条和页面交互 |
| 同步脚本 | `js/scraper.js` | 从店铺三个分类增量同步新品并生成 JSON/JS 数据 |
| 游戏数据 | `data/games.json` | 游戏与分类数据源 |

## 当前状态
